####git 

1. 初始化本地仓库;  git init 
   > 成功的标识 Initialized empty Git repository in E:/随堂/1820Acoding/1208/.git/
2.  git status 在git init 之后查看状态, 绿色的标识新增, 红色的标识已经更改;
3.  git commit -m "注释" 把暂存区里的内容提交到本地仓库;
4.  git reflog  查看版本日志;
5.  git reset --hard 版本还原;
6.  git --help 指令提示;
